About This Application

This application is built using Angular, Spring Boot, and MySQL database. It provides a comprehensive platform for conducting online examinations with separate interfaces for administrators and examinees.

Key Features:

  • Separate user modules for administrators and examinees
  • Category-based Exams organization
  • Exams publishing control
  • Detailed report cards in PDF format
  • Answer review functionality

Tech Stack:

Angular Spring Boot MySQL Java REST API
